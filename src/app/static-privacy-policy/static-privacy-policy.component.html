Privacy Policy

The Spartan Connect team values your privacy. For our users of the mobile app, no personal data will be collected at all. All user recommendations (favorite categories, selected grade level, etc.) are stored locally on the device. We ask our users on Android devices, however, to please note that the permissions granted upon installation are necessary for the app to run. No data is collected from or about the user using those permissions. We are working on removing them in the near future.

For our teachers using our web application (this site), some key information will be stored upon first login with a Google account, including:

  • Full name
  • Email address
  • Profile picture (from Google)

In addition, whenever a user logs in, our web application will store a new date and time. This is so that administrators can accurately determine the usage of Spartan Connect over time.

We hope that you enjoy using Spartan Connect.
